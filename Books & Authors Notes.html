How to crack this topic: it's mostly one-liners (book ↔ author). Focus on the ~15 most-repeated authors & their booklists (given at the end). For statement-based CGL questions, use elimination: claims that a book "won the Booker/Pulitzer/Sahitya Akademi" are usually wrong; and a book's central theme can only be ONE thing — so if two options describe different themes, only one is correct.

💡 Key Exam Tips — Yaad Rakho!
  • Memorize the author first, then their booklist — most questions are simple book ↔ author one-liners.
  • In statement / Assertion–Reason questions, claims that a book "won the Booker / Pulitzer / Sahitya Akademi" are usually the wrong/false part — verify carefully.
  • A book's central theme can only be ONE thing — if two options describe different themes, only one is correct.
  • Extreme keywords ("only," "exclusively," "never," "always") in a statement → usually wrong.
  • "NOT written by / NOT a work of" questions → know each author's real list so you can spot the odd one (e.g., The Great Derangement = Ghosh, not Roy).
  • First-of-a-kind facts repeat a lot: Arundhati Roy = first Indian woman Booker; Amartya Sen = first Indian Economics Nobel; Alice Munro = first Canadian Literature Nobel.
